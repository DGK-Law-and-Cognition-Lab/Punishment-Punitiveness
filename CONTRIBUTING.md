# Contributing

Thank you for your interest in this research project. This repository primarily serves as an open-science companion to our published research, so we welcome contributions in the following areas:

## Bug Reports
If you find errors in the analysis code, data inconsistencies, or broken links, please open a GitHub issue describing the problem with specific file references.

## Replication & Extension
We encourage replication attempts and extensions. If you use our data or code, please cite the original study (see CITATION.cff) and consider sharing your findings as an issue or discussion.

## Code Improvements
Pull requests for code efficiency improvements, better documentation, or additional visualizations are welcome. Please describe the change and its rationale.

## Questions
For questions about the data, methodology, or findings, please open a GitHub issue or contact the authors directly:
- Dan Simon: dsimon@law.usc.edu
